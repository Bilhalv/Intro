Uso de !+enter

  O uso de "!+enter" no VSCode é uma forma rápida de criar um bloco de código de inicialização, inserindo automaticamente uma estrutura básica do tipo de arquivo que está sendo criado. Detalhe, a funcionalidade só serve caso a extensão do arquivo seja declarada na criação do mesmo, assim como na foto.

Uso de shift+alt+baixo

  A combinação de teclas "shift+alt+baixo" no VSCode permite duplicar a linha atual para a linha abaixo, economizando tempo e esforço ao copiar linhas de código. Adicionalmente, funciona para seleções de paragrafos inteiros.

Uso de selecionar e arrastar

  No VSCode, é possível selecionar um trecho de código e arrastá-lo para cima ou para baixo com o mouse, permitindo reorganizar facilmente o código e alterar a ordem dos elementos de forma visual e intuitiva. Isso proporciona uma maneira conveniente de refatorar e ajustar o layout do código.

Uso de comentários

  Sempre recomendo o uso de comentários independente da linguagem utilizada para ajudar a catalogar o que está sendo feito em cada parte do código, especialmente em funções de JavaScript e CSS pela complexidade que os codigos podem acabar criando, então sempre tentar encontrar seu respectivo comando para comentário é estritamente necessário.
